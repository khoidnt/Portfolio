# 📸 BY KHOI DINH - Photography Portfolio

Chào mừng bạn đến với dự án website portfolio nhiếp ảnh cá nhân. Mục tiêu của dự án là mang đến trải nghiệm xem ảnh mượt mà, tối giản và tập trung vào cảm xúc thị giác.

## ✨ Tính năng nổi bật

- **Parallax Hero Section:** Hiệu ứng chiều sâu ở phần ảnh bìa giúp tạo ấn tượng ngay khi mở trang.
- **AOS Reveal Animation:** Ảnh xuất hiện mượt theo nhịp cuộn để tăng cảm giác kể chuyện.
- **Responsive Design:** Bố cục tối ưu trên mobile và desktop, giữ tỷ lệ hiển thị ảnh đẹp mắt.
- **Gallery theo chủ đề:** Ảnh được tổ chức thành nhóm giúp người xem khám phá dễ hơn.

## 🛠 Công nghệ sử dụng

- **Ngôn ngữ:** HTML5, CSS3 (CSS Variables), JavaScript thuần.
- **Thư viện:**
    - [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) cho hiệu ứng xuất hiện nội dung.
    - [Font Awesome](https://fontawesome.com/) cho hệ thống biểu tượng.
    - Google Fonts (Montserrat, Cormorant Upright) cho nhận diện thị giác.
- **Triển khai:** [Cloudflare Pages](https://pages.cloudflare.com/).

## 🚀 Chạy dự án local

Vì đây là website tĩnh, bạn có thể mở trực tiếp hoặc chạy local server:

1. Clone hoặc tải source code về máy.
2. Mở thư mục `Portfolio`.
3. Chạy bằng một trong các cách:
     - Mở trực tiếp `index.html` trong trình duyệt.
     - Hoặc dùng VS Code Live Server để có tự reload khi chỉnh sửa.

## 📁 Cấu trúc thư mục

```text
Portfolio/
├── images/              # Chứa toàn bộ ảnh portfolio
│   ├── chan-dung/
│   ├── doi-thuong/
│   ├── ngoai-canh/
│   └── nksc_param/
├── index.html           # File cấu trúc chính của website
├── style.css            # Style và responsive layout
├── script.js            # Xử lý tương tác và animation
├── robots.txt
├── sitemap.xml
└── README.md
```

## 📌 Ghi chú

- Tối ưu kích thước ảnh trước khi đưa vào thư mục `images/` để tăng tốc độ tải trang.
- Cập nhật `sitemap.xml` khi thêm các section hoặc trang mới.
- Có thể mở rộng thêm phần liên hệ hoặc form booking ở các phiên bản sau.
